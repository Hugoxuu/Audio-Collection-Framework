Question: What is Audio Collection Framework?
A framework that collects Human-Human and Human-Bot conversations in spoken/audio form, together with IVR data, on the platform of AWS Connect.
Question: Where can I find the code package?
The package is available at:
https://code.amazon.com/packages/AWSDeepSenseSpokenDataCollectionFramework/trees/mainline
Question: What services does the framework use?
The framework uses multiple AWS services, including:
AWS Connect, AWS Lex, AWS Lambda, AWS Dynamo DB, AWS S3, AWS IAM, AWS Kinesis Video/Data Streams, and AWS Transcribe.
